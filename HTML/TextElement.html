The Basic Language of the Web: HTML1

Posted by Laura Jones on Monday, June 21st 2027

All modern websties and web applications are built using three fundamental technologies: HTML, CSS and JavaScript

These are the languages of the web.

In this post, let's focus on the HTML. We will learn what HTML is all about, and why you too should learn it.

What is HTML

HTML stands for Hyper Text M arkup Language. It's a markup langauge that web developers user to structure and describe the content of a webpage (not a programming language).

HTML consists of elements that describe different types of content: paragraphs, links, headings, images, video, etc. Web browsers understand HTML and render HTML code as websites.

In HTML, each element is made up of 3 parts

  1. The opening tag
  2. The closing tag
  3. The actual element
